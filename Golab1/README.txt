При запуске программа выводит меню с номерами заданий.
Пользователь выбирает номер задания, и программа выполняет соответствующую функцию.
Каждая функция отвечает за выполнение одного задания.
Если пользователь выбирает "0", программа завершает работу.
Инструкция по запуску:
Убедитесь, что у вас установлен Go.
Создайте файл, например lab1.go.
Сохраните код в этот файл.
Запустите программу командой:
"go run lab1.go"
Выберите задание в меню и следуйте инструкциям программы.
